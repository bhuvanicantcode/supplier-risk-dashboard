# 🏭 Supplier Risk Scoring & Supply-at-Risk (SaR) Simulation

**Author:** Pallapolu Bhuvan Chandra — BITS Pilani  
**Stack:** Python | pandas | NumPy | matplotlib | Streamlit

## 📌 Overview
A pharmaceutical supply chain risk dashboard that scores 33 global vendors using Monte Carlo simulation adapted from **Credit Risk VaR (PD × LGD × EAD)** methodology.

## 🚀 Run Locally
```bash
pip install -r requirements.txt
streamlit run supplier_final.py
```

## 🗂️ Data Sources
- USAID SCMS Delivery History Dataset (public domain)
- World Bank WGI 2012–2022 (CC-BY 4.0)

## ⚙️ Risk Dimensions
OTD Risk | Lead Time Variability | Governance Risk | Concentration Risk | Logistics Complexity | Planning Risk
